                                                                            我的博客
 
◊
 
项目简介：我的博客左侧导航栏主要包括首页、文章、关于我、项目和留言板和下方的一些联系方式。使用到的技术主要如下：
 
◊
 
使用vue-cli进行搭建工程，vue-router实现页面的跳转，同时通过vuex来集中式存储管理项目中的组件的共享数据，利用vuex同步存值以及取值。
 
◊
 
本项目主要采用props、$emit以及事件总线来进行组件间的通信，其中博客详情页就是通过事件总线的方式来实现滚动滑轮时右侧导航栏实时更新。
 
◊
 
使用mock模拟后台数据，生成随机的数据并拦截Ajax请求，实现前后端分离。
 
◊
 
使用@wheel监测鼠标滚轮触发事件，@mousemove和@mouseleave实现首页鼠标拖动功能。
 
◊
 
使用v-lazy懒加载可以实现只优先加载页面可视区域的图片，从而使得页面加载更为流畅。
 
◊
 
使用mixins提供在Vue组件中复用功能的方法，与 vuex 的区别是：mixins可以定义共用的变量，在每个组件中使用，引入组件中之后，各个变量是相互独立的，值的修改在组件中不会相互影响。
 
◊
 
使用组件递归渲染右侧文章分类和目录的列表数据结构。
 
◊
 
项目github地址：https://github.com/AfgQS/myself-blog-system
 

                                                                         后台管理系统
 
◊
 
项目简介：后台管理系统是一个比较传统的项目，主要使用JavaScript、Vuex、ElementUI 组件库等技术，完成了系统的增、删、改、查以及权限分配等功能，具体如下：
 
◊
 
使用 vue-element-admin 后台集成方案中的基础模板进行功能的扩展和项目的二次开发，  并上传至 git 进行版本管理。
 
◊
 
使用 vue-router 完成前端路由配置，  采用 ElementUI 组件库结合其他组件进行页面的设计。
 
◊
 
在 Vuex 中 getters 建立了公共的访问属性，  在 computed 里使用...mapGetters 的方式使得页面可以 快捷访问数据。
 
◊
 
采用RBAC权限管理，相对于传统方案，RBAC提供了中间层Role（角色），实现了用户和权限点的分离，想对某个用户设置权限，只需要对该用户设置相应的角色即可，而该角色就拥有了对应的权限。
 
◊
 
根据父子级之间的 id 和 pid 关系，  采用递归算法将组织架构的列表型数据转换成了树形数据，  将组织 架构以树形金字塔式结构呈现在页面上。
 
◊
 
项目github地址：https://github.com/AfgQS/bg_manage
